How it works

A supervisor orchestrating six specialists

This isn't a single prompt. A supervisor agent plans the work, dispatches each specialist, runs a reflection loop on weak output, and lets the Risk Manager veto the final call.

01
🛰️

Market Data

Uses real LLM tool-calling — the model autonomously decides to fetch live price, P/E, beta and news via function calls.

02
📊

Fundamental Analyst

Reasons over valuation, growth and margins to form a bullish / neutral / bearish lean.

03
📄

SEC Filings (RAG)

Fetches the latest 10-K from EDGAR, retrieves the most relevant passages, and grounds its analysis in primary source with citations.

04
📰

News & Sentiment

Scores recent headlines and flags material legal or regulatory risks.

05
🛡️

Risk Manager

Blends a deterministic quant score with LLM judgment — and can veto a recommendation outright.

06
✍️

Report Writer

Synthesizes every signal into a concise, risk-aware investment memo with a clear BUY / HOLD / SELL call.

Architecture

A hand-rolled, LangGraph-style state machine streams every transition to this UI over Server-Sent Events.
